Sample application that works with <b>Dagger 2</b> and <b>Butter Knife</b>.

<a href="https://google.github.io/dagger/">Dagger 2</a> is dependency injection framework for Android.
And <a href="http://jakewharton.github.io/butterknife/">Butter Knife</a> is library that accomplishes field and method binding for Android views
which uses annotation processing to generate boilerplate code for you.
